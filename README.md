# Installing the application
To use and build the application, you need to have Microsoft Visual Studio 2019. The following steps should be followed; ->Download the code ->Open QuarterlySales.sln file which will open the project in visual studio ->Right click on the solution and rebuild. ->Run the solution by clicking the run button. ->Visual studio will package and run the project using your default browser
